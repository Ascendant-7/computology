# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-31

### Added

- **Authentication Module:** Login, Register, Forgot Password screens with validation.
- **Home Module:** Banner carousel, category chips, and featured product grid.
- **Profile Module:** Profile view, edit profile, and theme (Light/Dark) toggle.
- **Architecture:** Provider state management, `SharedPreferences` for persistence, and centralized mock data services.
- **Design:** Material 3 implementation throughout the app.
